## 角色和权限：

系统管理员 > 管理员 > 其他.

系统管理员：是指内置的一个管理员，它无视权限校验，全部功能都可用，只能操作数据库的字段才能指定。
管理员：有管理权限 + 用户自身权限
普通用户：用户自身权限
其他：...

所以系统必须至少内置一个系统管理员，以及初始化一个普通用户角色。
